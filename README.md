# [简体中文](./README_ZH.md)
Easy to use and lightweight gis tool for web

# Description
EasyGIS is based on [Openlayers](https://openlayers.org), can help you finish some easily features.

# Features
- Vector data preview
- Geometry format and coordinate system transform
- Draw feature
- OGC layers preview
